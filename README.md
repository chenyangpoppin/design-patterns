<h2>Java Design Patterns</h2>
<p>
  An implementation of design patterns with Java code. The design patterns include Builder, Abstract Factory, Adapter, Bridge, Singleton, Chain, DAO, Fascade, Dependency Injection etc. These design patterns can be divided into three categories as follows:
</p>

<h3>Creational Patterns</h3>
<ul>
  <li>Abstract factory</li>
  <li>Builder</li>
  <li>Factory method</li>
  <li>Prototype</li>
  <li>Singleton</li>
</ul>

<h3>Structural Patterns</h3>
<ul>
  <li>Adapter</li>
  <li>Bridge</li>
  <li>Composite</li>
  <li>Decorator</li>
  <li>Facade</li>
  <li>Flyweight</li>
  <li>Proxy</li>
</ul>

<h3>Behavioral Patterns</h3>
<ul>
  <li>Chain of responsibility</li>
  <li>Command</li>
  <li>Interpreter</li>
  <li>Iterator</li>
  <li>Mediator</li>
  <li>Memento</li>
  <li>Observer</li>
  <li>State</li>
  <li>Strategy</li>
  <li>Template</li>
  <li>Visitor</li>
</ul>
